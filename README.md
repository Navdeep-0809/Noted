# 📝 Noted - Real-Time Collaborative Workspace

**Noted** is a premium, full-stack collaborative application designed to showcase modern web development concepts. It allows multiple users to create notes, edit in real-time, and sketch together on a shared whiteboard.

---

## 🚀 Features

- **⚡ Real-Time Editor**: Multi-user rich text editing powered by Quill and Socket.io.
- **🎨 Collaborative Whiteboard**: Real-time sketching with shapes, colors, and export functionality.
- **👥 Smart Sharing**: Invite others by username to collaborate on your personal notes.
- **📊 Insights Dashboard**: Track productivity with detailed statistics on note creation, deletions, and collaborations.
- **💎 Premium UI**: A modern, dark-themed interface built with Glassmorphism and Outfit/Inter typography.

---

## 🎓 Academic Concepts Covered

This project is built to demonstrate core backend and full-stack architecture:

- **Middleware**: Implementation of Application-level, Router-level, and custom Error-handling middleware.
- **Authentication**: Secure registration with **Bcrypt** password hashing and **JWT** (JSON Web Token) session management.
- **Security**: Implementation of HTTP-only cookies and protected route patterns.
- **Communication**: Full-duplex, bidirectional communication using **Socket.io**.
- **Database Architecture**: NoSQL integration with **MongoDB** and **Mongoose** ODM for schema-driven data management.
- **Performance**: Demonstration of Blocking vs Non-Blocking I/O operations.

---

## 🛠️ Technical Stack

- **Backend**: Node.js & Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Real-Time**: Socket.io
- **Templates**: EJS (Server-Side Rendering)
- **Styling**: Vanilla CSS (Custom Glassmorphism System)
- **Auth**: JWT & express-session
